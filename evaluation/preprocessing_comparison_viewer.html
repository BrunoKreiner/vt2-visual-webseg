Comparison of Preprocessing Methods for Web Segmentation

Left: Original method - Each polygon within a multipolygon is processed separately, resulting in many small, fragmented bounding boxes.

Right: Improved method - The entire multipolygon structure is processed as a single unit, creating a single bounding box that encompasses all points from all polygons in the multipolygon.
